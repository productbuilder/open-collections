# Example Test Collection

This folder contains an MVP dataset for Open Collections Manager.

## Contents

- `collection.json`: inline manifest with item metadata and public media URLs.

## Dataset goals

- exercise card/grid browsing in the Collector UI
- test mixed metadata completeness
- test mixed rights/license values
- validate manifest export and Open Collections handoff shape

## Notes

- Media files are referenced by stable public placeholder image URLs to keep the repository lightweight.
- Item IDs are stable and not derived only from filenames.
- This dataset is meant for development and demo use.
