Register a collection

Add a collection or collection source to the Open Collections registry. Enter a URL, we check it, and if it is valid it can appear in the registry.

Submit a URL

How it works

  • Submit a single collection URL or a collections source URL.
  • Open Collections checks whether the URL is valid.
  • If it passes validation, it can be added to the registry.
  • Recent registered collections appear in the list above.
