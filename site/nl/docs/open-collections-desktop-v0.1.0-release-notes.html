Open Collections Desktop v0.1.0

Windows preview release.

Preview Windows v0.1.0

Download the latest desktop release from GitHub Releases

Overview

Open Collections Desktop is the first desktop release of Open Collections, combining the Manager and Browser in a single Windows app.

This is a preview release. Core host and publishing flows are available, but some advanced features and edge-case validation are still in progress.

Before announcing v0.1.0 publicly, maintainers should upload the Windows installer to GitHub Releases and complete the desktop release checklist.

Included in v0.1.0

  • Open Collections Manager
  • Open Collections Browser
  • built-in example browsing
  • local folder host support
  • GitHub host support
  • S3-compatible host support
  • remembered hosts and reconnect flows
  • desktop credential storage path
  • publish-state and host-state improvements

Known limitations

  • registry browsing is not included yet
  • custom domain hosting is not included yet
  • full sync/conflict-resolution workflows are not included yet
  • some host repair and reconnect flows still need broader real-world validation
