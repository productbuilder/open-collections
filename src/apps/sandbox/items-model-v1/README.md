# items-model-v1

Versioned sandbox presentation prototype for the **Items / Structure / Interfaces** concept.

## Purpose

- Demonstrate the concept as a slide-style browser presentation.
- Provide a reusable v1 baseline that can be iterated in-place and compared against future versions (for example `items-model-v2`).
- Explore a path toward future Present templates and website storytelling renderers.

## Scope

This prototype is intentionally lightweight and self-contained:

- `index.html` for app shell and presentation styling
- `data.js` for scene content and sample item data
- `app.js` for rendering, hash navigation, and keyboard controls
