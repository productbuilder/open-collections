Get started (technical companion)

Use this with the public Get started page when your technical team needs deeper implementation detail.

Collection outputs

  • collection.json at a stable URL
  • stable item and media URLs
  • optional domain discovery file at /.well-known/source.json

Practical flow

  1. Start from the source you already have: local files, existing media URLs, or exported records from spreadsheets and other systems.
  2. Open Collection Manager and create or update collection data.
  3. Add metadata, confirm item links, and verify public media URLs where you are referencing externally hosted assets.
  4. Host the collection files and verify the published URLs.

Referenced media example

Create a collection for a small exhibit, add two item records, and point each item to an existing public image URL. Add titles, descriptions, and rights details in the collection metadata, then publish collection.json so the collection can be shared without moving the media files to a new host.
