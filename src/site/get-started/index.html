Get started with the materials you already have

Use this page for a calm, practical first run. You can begin with local files, existing media links, or exported records and move toward a shareable collection in a few steps.

1. Choose a starting point

  • Gather the files, folders, links, or exported records that belong in one collection.
  • Use clear titles or file names and remove anything you do not want to share.
  • Keep enough context together that browsing will feel understandable later.

2. Open Collection Manager

Open Collection Manager in your browser and start a new collection for the material you want to organize.

Open Collection Manager

3. Add items from files, links, or records

Some collections start from a folder on disk. Others begin with metadata from a spreadsheet or with items that point to media already hosted on another site. Use the path that matches what you already have.

4. Add titles and details

Add a collection title, short description, and item details that help people understand what they are viewing, whether the media is uploaded locally or referenced from an existing URL.

5. Simple example: publish metadata with referenced media

Create a collection, add two items, and use existing public image URLs for the media. Then add titles and short descriptions so the collection can be published and shared while the media stays on its current host.

6. Save and share

Save your collection, then share the URL with your team or audience. You can keep improving the records, media links, and presentation over time.
